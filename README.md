# Jupyter-Notebook
Jupyter Notebook使用


### jupyter notebook卸载重装

```
For Python 2
sudo pip install --upgrade --force-reinstall --no-cache-dir jupyter

For Python 3
sudo pip3 install --upgrade --force-reinstall --no-cache-dir jupyter

这应该重新安装PyPi中的所有内容。这应该解决这个问题，
```
